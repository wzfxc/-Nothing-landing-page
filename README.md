# Nothing — E-Commerce Landing Page

This is a modern, responsive landing page for the "Nothing" tech brand, showcasing their unique products (Phone (1), Ear (2), and accessories). The project is built with a focus on clean layout architecture, high performance, semantic HTML, and smooth user interactions.

## Technologies Used

* **HTML5** — Semantic layout structure (`<header>`, `<main>`, `<section>`, `<article>`).
* **SCSS / BEM** — Organized, modular, and scalable styling architecture following BEM methodology.
* **JavaScript (ES6)** — Dynamic logic for navigation menus and form interactions.
* **AOS (Animate On Scroll)** — Lightweight library for smooth scroll-driven animations.
* **Google Fonts** — Integrated web typography (Space Grotesk & Space Mono).

## Getting Started

Follow these simple steps to clone the repository and run the project locally:

### 1. Clone the repository
```bash
git clone https://github.com/wzfxc/-Nothing-landing-page.git
cd <project-name>
```
### 2. Install dependencies
```bash
npm install
# or
yarn install
```

### 3. Run the project locally
```bash
npm start
```
